# Vue

Vue es el framework que empezamos a usar para nuestro nuevo frontend.

Antiguamente, todas las vistas fueron creadas en el motor de plantillas de Blade (algunas siguen funcionando a día de hoy), pero desde Octubre de 2025, implementamos Vue para nuevas vistas y para, poco a poco, refactorizar vistas antiguas en Vue.

Actualmente trabajamos en la versión `^3.5.22`

- Documentación oficial: https://vuejs.org/guide/introduction.html
