# Laravel

Laravel es el framework base de nuestro backend.

Actualmente trabajamos con la versión `12.42.0`.

- Documentación oficial: [https://laravel.com/docs/12.x](https://laravel.com/docs/12.x)

## Arquitectura MVC

Intentamos estructurar Laravel bajo el patrón **MVC** (**Modelo-Vista-Controlador**), para conseguir separar correctamente todas las distintas responsabilidades y haciendo el código más escalable y mantenible:

- [Modelos](/laravel/modelos)
- Vistas
- Controladores

## Servicios
